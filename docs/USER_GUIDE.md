# SW 점령전 사용자 가이드

이 문서는 `SW 점령전` 웹사이트를 사용하는 길드장, 부길드장, 길드원을 위한 간단한 사용법입니다.

---

## 1. 계정 가입과 로그인

### 길드장 가입

1. 시작 화면에서 `길드장 가입`을 선택합니다.
2. 아이디, 비밀번호, 길드 이름, 인게임 닉네임, 이메일을 입력합니다.
3. 가입 요청을 보내면 관리자 승인 전까지 로그인할 수 없습니다.
4. 관리자가 승인하면 길드가 생성되고, 가입한 계정은 해당 길드의 길드장이 됩니다.

### 길드원 가입

1. 시작 화면에서 `길드원 가입`을 선택합니다.
2. 가입하려는 길드 이름과 계정 정보를 입력합니다.
3. 길드장 또는 부길드장이 승인하기 전까지 로그인할 수 없습니다.
4. 승인 후 길드에 소속된 상태로 서비스를 이용할 수 있습니다.

### 기존 계정의 길드 가입

이미 계정은 있지만 소속 길드가 없는 경우, 로그인 후 길드 가입 요청 화면만 사용할 수 있습니다.

- 가입 요청 중에는 다른 길드에 중복 요청할 수 없습니다.
- 요청을 취소하면 다시 다른 길드에 가입 요청할 수 있습니다.

---

## 2. 권한별 사용 범위

| 권한 | 가능한 작업 |
|---|---|
| 길드장 | 길드원 관리, 등급 변경, 추방, 길드장 양도, 더미 계정 관리, 모든 인벤토리/방덱 관리 |
| 부길드장 | 가입 승인, 더미 계정 관리, 길드원 인벤토리/방덱 관리 |
| 길드원 | 본인 인벤토리와 본인 방덱 관리, 전투 연구 작성 |

---

## 3. 인벤토리 탭

길드원이 보유한 몬스터 수량을 관리하는 화면입니다.

1. 관리할 길드원을 선택합니다.
2. 성급 필터와 속성 필터로 몬스터를 찾습니다.
3. 몬스터별 보유 수량을 입력합니다.
4. 저장 버튼을 눌러 반영합니다.

수량 제한:

- 몬스터별 최대 수량은 10마리입니다.
- 직접 10보다 큰 숫자를 입력하면 10으로 보정됩니다.
- 음수를 입력하면 0으로 보정됩니다.
- 이미 방덱에 사용 중인 수량보다 낮게 줄일 수 없습니다.

---

## 4. 방덱 탭

길드원 인벤토리를 기준으로 실제 생성 가능한 방덱을 등록하는 화면입니다.

1. 방덱 소유 길드원을 선택합니다.
2. 몬스터 3마리를 선택합니다.
3. 첫 번째 몬스터가 리더로 처리됩니다.
4. `방덱 생성`을 누르면 해당 길드원의 방덱으로 등록됩니다.

참고:

- 방덱 소유 길드원을 바꾸면 선택한 몬스터 칸이 초기화됩니다.
- 동일한 몬스터 3마리 조합은 하나로 묶여 표시됩니다.
- 펼치기를 누르면 해당 방덱을 보유한 길드원을 확인할 수 있습니다.

---

## 5. 길드 방덱 탭

길드 공용으로 참고할 방덱을 등록하는 화면입니다.  
특정 길드원 소유가 아니라, 길드 전체가 참고하는 템플릿에 가깝습니다.

주요 기능:

- 길드 공용 방덱 등록
- 생성 가능한 길드원 수 확인
- 상세 보기에서 생성 가능 길드원 목록 확인

탈퇴한 길드원은 생성 가능 길드원 계산에서 제외됩니다.

---

## 6. 전투 연구 탭

방덱 공략, 공격 조합, 전투 결과 메모를 공유하는 게시판입니다.

게시글 작성:

1. 연구 대상 방덱 3마리를 선택합니다.
2. 제목과 연구 내용을 작성합니다.
3. 등록 버튼을 누릅니다.

댓글 작성:

- 게시글 상세에서 댓글과 공격덱을 함께 남길 수 있습니다.

제한:

- 제목: 최대 100자
- 본문: 최대 3000자
- 댓글: 최대 1000자
- 게시글과 댓글은 각각 30초에 한 번 작성할 수 있습니다.
- 작성 실패 시 입력하던 내용은 유지됩니다.
- 게시글과 댓글은 10개 단위로 페이지가 나뉩니다.

---

## 7. 회원 관리 탭

길드장과 부길드장이 사용하는 관리 화면입니다.

### 가입 승인

- 길드원 가입 요청을 승인하거나 거절할 수 있습니다.

### 길드 멤버

- 길드원의 등급과 최근 접속일을 확인할 수 있습니다.
- 길드장은 등급 변경, 추방, 길드장 양도를 할 수 있습니다.

### 더미 계정

로그인 계정 없이 인벤토리와 방덱만 관리할 가상 길드원을 만들 수 있습니다.

- 더미 계정도 길드 인원 35명 제한에 포함됩니다.
- 길드원이 모두 가입하기 전에도 길드장이 미리 방덱을 정리할 때 사용할 수 있습니다.

### 재가입 불가 목록

추방된 길드원은 재가입 불가 목록에 들어갑니다.  
길드장은 필요할 경우 재가입 제한을 해제할 수 있습니다.

---

## 8. 내 정보 탭

내 계정 정보를 확인하고 닉네임 변경 요청 또는 길드 탈퇴를 할 수 있습니다.

확인 가능한 정보:

- 아이디
- 이메일
- 현재 닉네임
- 소속 길드

닉네임 변경은 바로 반영되지 않고 관리자 승인 후 반영됩니다.

일반 길드원과 부길드장은 길드 탈퇴가 가능합니다.  
길드장은 먼저 길드장을 다른 길드원에게 양도해야 탈퇴할 수 있습니다.

---

## 9. 사용 팁

- 길드장은 실제 길드원이 모두 가입하기 전 더미 계정으로 인벤토리와 방덱을 먼저 정리할 수 있습니다.
- 전투 연구는 리더 효과, 포함 몬스터, 4성 방덱 필터를 활용하면 원하는 공략을 빠르게 찾을 수 있습니다.
- 인벤토리 수량은 방덱 생성 가능 여부와 직접 연결되므로 실제 보유 수량에 맞게 관리하는 것이 좋습니다.
